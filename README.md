## 📌 What chimere-odo does

chimere-odo is a Windows app that helps you send requests through the right path.

It can:
- Classify the intent of a request
- Route work to the right compute path
- Use a quality gate before it returns results
- Run web search as part of the process
- Store useful context in memory layers
- Improve output from one pass to the next

This makes it useful when you want one place to manage search, memory, and model use.

## ⚙️ Basic use

chimere-odo is built to handle requests in a simple flow:

1. Type or paste your request
2. Let the app classify the intent
3. The app chooses the best route for the task
4. It checks the output with a quality gate
5. It may search the web if the task needs fresh info
6. It stores useful context in memory for later use

This helps when you want the app to decide how much work a request needs.

## 🧠 Memory layers

chimere-odo uses more than one memory tier:
- Short-term memory for the current task
- Working memory for active context
- Longer-term memory for repeated facts or user patterns

This can help the app keep track of what matters across steps without making you repeat yourself.

## 🌐 Web search flow

When a request needs current information, chimere-odo can use a web search pipeline.

It is useful for:
- Recent facts
- Product details
- News-related tasks
- Pages that change often

You can treat it like a built-in search step that supports the main task instead of replacing it.

## ✅ Quality gate

The quality gate checks the output before the app returns it.

It helps the app:
- Catch weak results
- Retry when needed
- Choose a better path
- Keep output more useful

If a result does not meet the set bar, the app can try again with a different route.

## 🔀 Adaptive compute routing

chimere-odo can route simpler tasks and harder tasks in different ways.

For you, that means:
- Fast handling for easy requests
- More effort for difficult requests
- Better use of available compute
- Less waste on small tasks

You do not need to manage this by hand. The app makes the choice during the run.
